
# 🚗 EV Mechanic AI Agent

An intelligent AI system that diagnoses Electric Vehicle (EV) issues using NLP and clustering techniques.

---

## 🔥 Features

- Detects multiple issues from a single input  
- Handles noisy and unstructured text  
- Uses BERT for semantic understanding  
- Fine-grained clustering (20+ issue types)  
- Confidence-based decision making  
- Interactive follow-up questioning (like a real mechanic)  

---

## 🧠 Tech Stack

- BERT (sentence-transformers)  
- MiniBatchKMeans (clustering)  
- Python  
- pandas, numpy, scikit-learn  

---

## ⚙️ How to Run

1. Install dependencies:  
   pip install -r requirements.txt  

2. Open the notebook:  
   ev_mechanic_agent.ipynb  

3. Run all cells  

---

## 💡 Example

Input:  
battery overheating and motor noise  

Output:  
🔍 Issues Detected:

- motor vibration  

⚠️ Confidence: 0.62

- Inspect battery cooling system
